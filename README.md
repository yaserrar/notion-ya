### Prerequisites

**Node version 18.x.x**

### Cloning the repository

```shell
git clone https://github.com/yaserrar/notion-ya
```

### Install packages

```shell
npm i
```

### Setup .env file

```js
DATABASE_URL=

GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET_ID=

NEXT_AUTH_SECRET=

NEXT_PUBLIC_LINK=
```

### Start the app

```shell
npm run dev
```
